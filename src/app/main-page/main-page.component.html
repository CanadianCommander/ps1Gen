PS1 Editor

Sex up your terminal with a new PS1 string!
This editor allows you to easly edit the PS1 string that produces your terminal prompt, allowing you to produce amazing PS1 strings.

Instructions

Use the button to add new elements to your PS1 string. (tab also does this)
Click on an element to edit its text.
Use the to drag and drop elements, reordering them.
Use the to change foreground and background color
Use the to select the "content type" of the element. i.e. text decoration / terminal escape codes
Finally use the to delete elements.
When you are done editing your string simpily copy and paste the PS1 Code at the bottom of the page in to your terminal.
